# Start Bootstrap Creative

Landing page project based on the Creative template.

## Live Structure

- `index.html`
- `css/styles.css`
- `js/scripts.js`
- `assets/` (images and favicon)

## Run Locally

This is a static website, so you can run it by opening `index.html` directly in your browser.

For a better local development experience, use a simple local server (for example with VS Code Live Server).
